NITC HOSTEL RESERVATION SYSTEM

This is a web application for the reservation of rooms in hostels and guest house.This allows the students to reserve rooms without having to be present physically. The web application allows students to choose the rooms of their choice.The allotment is done in a first-come first-serve basis.The details like availablity of rooms,location etc. would be displayed. This reduces the confusion the students might be having while coming for hostel room allocation.
